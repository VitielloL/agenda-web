# 📅 Agenda do Vit

Projeto de uma **agenda de tarefas estilo calendário**, desenvolvido com Next.js, TypeScript e Tailwind CSS.

O objetivo é praticar componentização, organização de estado e UI moderna com React.

---

## 🚀 Tecnologias

- Next.js
- React
- TypeScript
- Tailwind CSS

---

## 📌 Funcionalidades

- Organização de tarefas por dia
- Marcação de tarefas como concluídas
- Interface moderna e responsiva
- Componentização reutilizável
- Dados mockados para simulação de agenda

---

## 🧱 Estrutura

- **DayColumn** → coluna de cada dia da agenda
- **TaskCard** → card individual de tarefa
- **mock** → dados simulados
- **types** → tipagem das tarefas

---

## 🎯 Objetivo

Projeto criado para estudo de:

- React com TypeScript
- Componentização
- Estilização com Tailwind
- Estrutura de projetos Next.js
